# 音色克隆使用指南

## 概述

小易智能助手现已集成 **CosyVoice3** 音色克隆服务，支持零样本音色克隆。

## 本地模型

你本地已安装以下语音克隆模型：

### 1. CosyVoice3 (推荐) 🌟
- **位置**: `C:\E\Fun-CosyVoice3-0.5B`
- **特点**: 零样本3秒克隆，质量最高
- **语言**: 中英日韩德法西意俄 + 18种中国方言
- **启动**: 运行 `Start-All-Services.bat`

### 2. VITS-Umamusume
- **位置**: `C:\E\VITS-Umamusume-voice`
- **特点**: 赛马娘角色音色，二次元风格

## 启动服务

### 方式一：一键启动（推荐）
双击运行：
```
Start-All-Services.bat
```

这将同时启动：
1. CosyVoice API 服务 (端口 50000)
2. 小易智能助手 (端口 3000)

### 方式二：单独启动 CosyVoice
```bash
cd C:\E\Fun-CosyVoice3-0.5B
.\env\python.exe start_api.py --port 50000
```

## 使用方法

### 1. 音色克隆
1. 打开小易助手页面 http://localhost:3000
2. 上传 3-30 秒的音频样本
3. 输入音频对应的文本
4. 系统自动克隆音色

### 2. 语音合成
使用克隆的音色合成语音：
```
POST http://localhost:50000/synthesize
{
  "text": "要合成的文本",
  "prompt_audio": "base64编码的参考音频",
  "prompt_text": "参考音频对应文本"
}
```

### 3. API 接口

| 端点 | 方法 | 说明 |
|------|------|------|
| `/health` | GET | 检查服务状态 |
| `/synthesize` | POST | 语音合成 |
| `/clone` | POST | 音色克隆 |
| `/voices` | GET | 列出所有音色 |
| `/languages` | GET | 支持的语言列表 |

## 注意事项

1. **Edge-TTS** 是预设音色，不支持克隆
2. **CosyVoice** 支持零样本克隆，3秒音频即可
3. 音频样本建议：
   - 时长：3-30秒
   - 格式：WAV (16kHz+)
   - 内容：清晰的语音，无背景噪音
   - 文本：提供准确的转录文本

## 服务优先级

小易会自动选择可用的最优服务：
1. CosyVoice (最佳音色克隆)
2. GPT-SoVITS
3. Edge-TTS (预设音色)
