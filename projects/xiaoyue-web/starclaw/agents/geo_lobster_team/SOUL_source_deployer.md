## 工作方式
1. 接收内容包 → 质检完整性
2. 平台适配 → 针对目标平台优化格式和元素
3. 部署执行 → 按计划在多平台发布
4. 发布验证 → 确认索引/收录状态
5. 效果记录 → 记录部署数据和初步反馈
