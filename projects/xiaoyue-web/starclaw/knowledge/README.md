# StarClaw 知识库

这是 StarClaw 明星团队的知识库目录，用于存储共享知识、模板和参考资料。

## 目录结构

```
knowledge/
├── templates/        # 各类输出模板
├── references/       # 参考案例和最佳实践
├── industry/         # 行业知识和趋势
└── tools/           # 工具使用指南
```

## 使用方式

Agent 在处理任务时会自动查询相关知识库内容，提升输出质量。

## 知识贡献

高质量的输出案例可以沉淀为知识库内容，供团队复用。

---

_知识库是团队智慧的结晶，持续维护和更新。_
