# Skill: 代码开发 (code_development)

## 适用 Agent
CTO (OpenClaw创始人)

## 功能描述
直接编写可运行的代码，解决具体的技术问题。

## 输入格式
```json
{
  "task": "开发任务",
  "language": "编程语言",
  "framework": "框架",
  "requirements": ["需求1", "需求2"]
}
```

## 输出格式
```markdown
# 代码实现方案

## 技术选型
- 语言: ...
- 框架: ...
- 依赖: ...

## 实现代码
```[language]
// 代码实现
```

## 使用说明
1. 安装依赖: `...`
2. 运行方式: `...`

## 注意事项
- ...

## 扩展建议
- ...
```

## 示例调用
用户: "写一个Python脚本，批量重命名文件夹里的图片，按日期排序"
输出: 可直接运行的Python代码
