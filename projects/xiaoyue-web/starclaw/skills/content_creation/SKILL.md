# Skill: 内容创作 (content_creation)

## 适用 Agent
周星驰(喜剧)、胡歌(深度)、任嘉伦(偶像)

## 功能描述
根据需求创作各类内容，包括剧本、文案、脚本等。

## 输入格式
```json
{
  "type": "script|copy|story|post",
  "topic": "主题",
  "style": "风格",
  "length": "篇幅",
  "platform": "发布平台"
}
```

## 输出格式

### 短视频脚本
```markdown
# [标题]

## 角色
- 角色A: 描述
- 角色B: 描述

## 场景
[场景描述]

## 分镜脚本
| 镜号 | 画面 | 台词 | 时长 |
|------|------|------|------|
| 1 | ... | ... | 3s |
| 2 | ... | ... | 5s |

## 金句/爆点
- ...

## 发布建议
- 话题标签: ...
- 发布时间: ...
```

### 营销文案
```markdown
# [标题]

## 主文案
[正文]

## 金句
[可单独传播的短句]

## 发布建议
- 平台适配: ...
- 配图建议: ...
```

## 示例调用
用户: "帮我写一个30秒的抖音搞笑短剧，主题是程序员相亲"
输出: 完整的分镜脚本
