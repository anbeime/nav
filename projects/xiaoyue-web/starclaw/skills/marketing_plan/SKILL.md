# Skill: 营销策划 (marketing_plan)

## 适用 Agent
雷军、杨幂、侯明昊、泰勒

## 功能描述
为产品或活动制定完整的营销策划方案。

## 输入格式
```json
{
  "product": "产品/活动名称",
  "target": "目标用户",
  "budget": "预算范围",
  "timeline": "时间周期",
  "goals": ["目标1", "目标2"]
}
```

## 输出格式
```markdown
# [产品/活动]营销策划方案

## 一、市场分析
### 目标用户画像
### 竞品分析
### 市场机会

## 二、策略定位
### 核心卖点
### 传播主题
### 品牌调性

## 三、执行方案
### 阶段一: 预热期
- 动作: ...
- 平台: ...
- KPI: ...

### 阶段二: 爆发期
- 动作: ...
- 平台: ...
- KPI: ...

### 阶段三: 长尾期
- 动作: ...
- 平台: ...
- KPI: ...

## 四、预算分配
| 项目 | 金额 | 占比 |
|------|------|------|
| ... | ... | ... |

## 五、风险预案
### 风险点
### 应对措施

## 六、效果预估
- 曝光量: ...
- 转化率: ...
- ROI: ...
```

## 示例调用
用户: "新APP上线，预算30万，3个月推广周期，目标是获取10万用户"
输出: 完整的营销策划方案
