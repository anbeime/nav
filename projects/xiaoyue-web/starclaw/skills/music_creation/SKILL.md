# Skill: 音乐创作指导 (music_creation)

## 适用 Agent
周杰伦

## 功能描述
为项目提供配乐、音效、音乐的创意方案。

## 输入格式
```json
{
  "project": "项目名称",
  "type": "bgm|soundtrack|jingle|sfx",
  "mood": "情绪基调",
  "duration": "时长"
}
```

## 输出格式
```markdown
# 音乐创意方案

## 音乐定位
- 风格: ...
- 情绪: ...
- 节奏: ...

## 配器建议
- 主奏乐器: ...
- 节奏组: ...
- 氛围层: ...

## 结构设计
- 前奏(0-5s): ...
- 主歌(5-25s): ...
- 副歌(25-45s): ...
- 尾声(45-50s): ...

## Suno 提示词
```
Style: [风格标签]
Mood: [情绪标签]
Instruments: [乐器]
Lyrics:
[歌词内容]
```

## 参考曲目
- [歌曲名] - [艺人]: [参考点]

## 应用建议
- 入点: ...
- 循环: ...
- 出点: ...
```

## 示例调用
用户: "给一个30秒的科技产品广告配乐，要现代感强"
输出: 完整的配乐方案 + Suno提示词
