# Changelog

All notable changes to this skill will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- 初始版本发布
- 支持17个搜索引擎 (8个国内 + 9个国际)
- 国内引擎: 百度、必应中国、必应国际、搜狗、360搜索、神马、头条搜索、知乎搜索
- 国际引擎: Google、Bing、DuckDuckGo、Brave、Startpage、Qwant、Yandex、Naver、WolframAlpha
- 支持高级搜索操作符: site:, filetype:, intitle:, inurl:, -, OR
- 支持时间过滤: 过去1小时/天/周/月/年
- 支持DuckDuckGo Bangs快捷搜索
- 支持WolframAlpha知识计算
- 添加高级搜索指南参考文档
- 添加国际搜索引擎指南参考文档

### Features
- 无需API密钥
- 隐私搜索引擎支持
- 多语言搜索支持
- 区域化搜索引擎支持
