# Skill: 战略决策 (strategic_decision)

## 适用 Agent
CEO 马斯克

## 功能描述
面对复杂的商业决策，用第一性原理拆解问题，输出可执行的战略方案。

## 输入格式
```json
{
  "task": "决策任务描述",
  "context": "背景信息",
  "constraints": ["约束条件1", "约束条件2"]
}
```

## 输出格式
```markdown
# 战略决策报告

## 问题本质
[用第一性原理拆解]

## 方案选项
### 方案A: [名称]
- 优势: ...
- 风险: ...
- 资源需求: ...

### 方案B: [名称]
- ...

## 推荐方案
[决策建议]

## 执行路径
1. 第一阶段: ...
2. 第二阶段: ...
3. 第三阶段: ...

## 成功指标
- [ ] 指标1
- [ ] 指标2
```

## 示例调用
用户: "公司要在下个季度做一场大型发布会，预算50万，怎么规划？"
输出: 完整的发布会战略规划报告
