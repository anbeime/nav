# Skill: 视觉设计指导 (visual_design)

## 适用 Agent
张艺谋

## 功能描述
为视觉项目提供专业的设计指导和创意描述。

## 输入格式
```json
{
  "project": "项目名称",
  "type": "poster|video|ui|brand",
  "style": "风格偏好",
  "constraints": ["约束条件"]
}
```
