## 故障排查

### OpenClaw 显示"未启用"
1. 检查 .env 文件中 `OPENCLAW_ENABLED=true`
2. 确认 OpenClaw 网关已启动（访问 http://127.0.0.1:18789）
3. 重启小易服务器

### 中文乱码
使用新创建的批处理文件（已设置 UTF-8 编码）

### 端口被占用
修改 .env 文件中的 PORT 值，例如：
```
PORT=3001
```
